# 01. 기존 코드에서 현재 목표로

## 출발점

원래 프로젝트는 `XWR1843 + DCA1000` 조합에서 raw ADC를 받아 실시간 `RDI / RAI`를 그리는 데모에 가까웠습니다.

기존 특징:
- `1TX / 4RX` 전제
- 실시간 시각화 중심
- 객체 검출이나 추적 로직은 거의 없음
- 하드코딩된 값이 많아 다른 보드로 옮기기 어려움

## 현재 목표

현재 목표는 아래와 같습니다.

- `IWR6843ISK-ODS + DCA1000`
- `3TX / 4RX`
- 실내 공간에서 여러 사람의 움직임을 실시간으로 추적
- 초기 단계에서는 정적 대상보다 `움직이는 사람`을 우선적으로 검출

## 왜 구조를 바꿔야 했는가

단순 RDI/RAI 표시만으로는 실제 추적이 되지 않습니다.  
사람 추적을 하려면 최소한 아래 단계가 필요합니다.

- raw ADC 수신
- radar cube 구성
- moving target 성분 강조
- 후보 검출
- 가까운 후보 병합
- 프레임 간 연속 추적

또한 `3TX / 4RX`로 바뀌면서 `12 virtual antenna`를 다뤄야 했고, ODS의 넓은 시야각 때문에 근거리 clutter와 multipath 문제도 더 크게 드러났습니다.

## 단계별 접근 전략

이 프로젝트는 대략 아래 순서로 진행했습니다.

1. 실시간 입력과 시각화가 새 보드 기준으로 실제로 동작하는지 확인
2. 후보 검출과 clustering을 붙여 사람 후보를 줄이기
3. Kalman 기반 추적으로 연속성을 확보
4. 로그와 무결성 계층을 추가해 “왜 흔들리는지”를 보이게 만들기

즉, 처음부터 완성형 추적기를 만드는 대신 `입력 -> 검출 -> 추적 -> 무결성` 순서로 점진적으로 확장한 구조입니다.
