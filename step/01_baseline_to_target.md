# 01. Baseline To Target

원본 프로젝트 상태:
- `XWR1843 + DCA1000` 기준의 실시간 `RDI / RAI` 시각화 데모에 가까웠다.
- 객체 추적기라기보다 raw ADC를 받아 FFT 결과를 그려주는 구조였다.
- 기본 전제는 `1TX / 4RX`였다.

목표 시스템:
- `IWR6843ISK-ODS + DCA1000`
- `3TX / 4RX`
- 한 공간 내 여러 인물의 실시간 추적
- 초기 목표는 정적 물체가 아니라 `움직이는 사람` 위주 검출

핵심 차이:
- `1TX / 4RX`에서 `3TX / 4RX`로 바뀌면서 `12 virtual antenna` 구성이 필요해졌다.
- 단순 RDI/RAI 표시만으로는 부족해서 `검출 -> 군집화 -> 추적` 단계가 추가돼야 했다.
- ODS는 시야각이 넓어서 근거리 clutter, multipath, sidelobe 문제가 더 잘 드러난다.

정리:
- Phase 1은 `실시간 입력과 시각화가 실제 센서 기준으로 도는가`
- Phase 2는 `후보점을 사람 단위로 줄일 수 있는가`
- Phase 3는 `끊김 없이 track를 유지할 수 있는가`
